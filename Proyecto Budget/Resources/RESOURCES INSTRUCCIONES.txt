Este Folder solo ha de contener los archivos .PNG de las imagenes que se van a usar en todas las interfaces. Se va a utilizar
el icon set de Font Awesome para lograr una homogeneidad de las interfaces, cualquier otro icon set/icono a usar por favor anotarlo.

Todos los iconos a usar deben descargarse en tamano 64x64 o 128x128 o sus equivalentes e importarse en este folder.

- Font Awesome Icon Set: https://www.iconfinder.com/iconsets/font-awesome
- https://www.iconfinder.com/iconsets/flat-circle-content
